
# Broken-Down Cursor Prompts (Best Practice)

Paste **ONE prompt at a time** into Cursor.
Do NOT combine prompts.

---

## Prompt A: Layout & Navigation
Build a responsive layout with a navbar and footer using Next.js App Router,
Tailwind CSS, and shadcn/ui.
The site is for a social events planning business.

---

## Prompt B: Homepage
Create a homepage for a social events planning company.
Include hero section, event types grid, how-it-works steps,
testimonials preview, gallery preview, and call-to-action.
Use Framer Motion animations.

---

## Prompt C: Services & Event Pages
Create services and event type pages for weddings, birthdays,
baby showers, and graduations.
Each page should include descriptions, benefits, and CTAs.

---

## Prompt D: Supabase Backend
Design and implement Supabase schema and RLS policies
for event inquiries, gallery, and testimonials.
Configure Supabase for Next.js App Router.

---

## Prompt E: Booking System
Implement a booking and contact form using Next.js Server Actions.
Persist submissions to Supabase.
Handle loading, success, and error states.

---

## Prompt F: Gallery System
Build a gallery system that fetches images from Supabase Storage.
Allow filtering by event type.
Ensure responsive and performant image rendering.

---

## Prompt G: Final Polish
Add SEO metadata, Open Graph tags, accessibility improvements,
and UI polish using Framer Motion.
Ensure the site is production-ready.
